### Build a single-file IABGVOC Issue & Requirement Dashboard in HTML/CSS/JS using Plotly.js 2.35.2 and Poppins (Google Fonts, weights 400–800). Data is a pre-embedded JS variable `const DATA` — no file upload UI.
### Theme: Background #f0f4f8, white cards, border #e2e8f0, radius 16px, soft shadow, hover lift (-3px translateY). Severity palette: Blocker=#e11d48 rose, Critical=#d97706 amber, Major=#2563eb blue, Minor=#059669 green. FMEA palette: High=#e11d48, Mid=#d97706, Low=#059669.
### Header: Centered. Pill eyebrow "IABGVOC · DIADesigner" → Poppins 800 42px title "Issue & Requirement Dashboard" with "Dashboard" in blue → subtitle showing DATA.meta.generated_at → 60px blue-to-teal (#2563eb→#0891b2) gradient 4px divider.
### DATA structure (embedded as `const DATA = {...}`): meta={generated_at:string, total_issues:int, total_reqs:int}, kpi={open_issues, open_reqs, cb_open, fmea_high_open, fmea_threshold}, trend={years:int[], issue_created[], issue_closed[], req_created[], req_closed[]}, severity={labels:["Blocker","Critical","Major","Minor"], issue_open[], req_open[]}, region={labels:string[], issue_cb_open[], issue_open[]}, tags={labels:string[], cb_open[]}, version={labels:string[], issue_open[]}, fmea_tier={high,mid,low}, state={labels:string[], counts[]}.
### Stats bar (below divider): Green pulsing live-dot pill showing "Issue 總數 N　Req 總數 N" from DATA.meta.
### 4 KPI cards (4-col grid): colored left border (--c CSS var) + emoji icon + Poppins 800 30px value + sub-label text. Blue=Issue Open (open_issues / total_issues), Teal=Req Open (open_reqs / total_reqs), Rose=Critical/Blocker Open, Amber=FMEA High Open (label shows "FMEA ≥ " + fmea_threshold).
### Row 1 (2-col): Spline area line chart — Issue Created solid blue filled rgba(37,99,235,0.08), Issue Closed solid green filled rgba(5,150,105,0.08), Req Created dotted teal, Req Closed dotted violet; x=years integer + Severity donut hole 0.58 Issue open only using Severity palette colors, textinfo=label+value.
### Row 2 (2-col): Region horizontal bar sorted descending by cb_open (blue opacity gradient rgba(37,99,235,1→0.2), yaxis autorange reversed, left margin 160px) + Top-10 Tag horizontal bar sorted descending (blue-to-teal rgb gradient, yaxis autorange reversed, left margin 240px); both show value as outside text, xaxis title "C/B 開放數".
### Row 3 (3-col donuts, hole 0.58): Version distribution (12-color VC palette, percent labels, issue_open by version) + Req FMEA tier (High/Mid/Low using FMEA palette, label+value) + Issue State distribution (8-color EC palette, percent labels).
### All Plotly: paper_bgcolor transparent, plot_bgcolor transparent, #e2e8f0 grid, Poppins font, displayModeBar false, responsive true. Staggered fadeUp animations (opacity 0→1, translateY 20px→0, .5s ease) on all cards with animation-delay .1s–1.1s steps.
